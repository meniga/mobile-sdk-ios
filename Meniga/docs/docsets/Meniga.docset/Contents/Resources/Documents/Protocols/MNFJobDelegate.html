MNFJobDelegate

@protocol MNFJobDelegate <NSObject>

A protocol for a job that can be used to access the jobs completion with a delegate.

  • -job:didCompleteWithResult:metaData:error:

    @abstract A method for handling a job completion. @discussion Requests are performed using NSURLSession which normally uses a background thread to perform it’s requests. Therefore this method is performed on the current thread the request is performed on. - parameter: result The result of the request. - parameter: metaData The metaData of the request. - parameter: error The error of the request.

  • -job:didCompleteOnMainThreadWithResult:metaData:error:

    @abstract A method for handling a job completion on the main thread. @discussion Requests are performed using NSURLSession which normally uses a background thread to perform it’s requests. This method forces the execution of completion to be performed on the main thread and is useful when handling ui action upon completion. - parameter: result The result of the request. - parameter: metaData The metaData of the request. - parameter: error The error of the request.
