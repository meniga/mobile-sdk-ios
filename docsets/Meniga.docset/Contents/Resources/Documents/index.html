Installation

Make sure you have the latest version of the Xcode command line tools installed:

xcode-select --install

Install fastlane using

[sudo] gem install fastlane -NV

or alternatively using brew install fastlane

Available Actions

iOS

ios deploy

fastlane ios deploy

Deploy MenigaSDK to cocoapods trunk Steps:

  • Bump version in podspec
  • Commit podspec changes
  • Create version tag
  • Push commit and tag to remote
  • Push podspec to Trunk specs
  • Generate documentation Options:

    Version

    version - new version number to release

ios tests

fastlane ios tests

Run tests: - Validate podspec - run-clang-format - to check Objective-c files formatting - synx - to check if xcodeproj has proper group structure - Mobile-ios-sdk unit tests
